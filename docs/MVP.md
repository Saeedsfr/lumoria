# Lumoria — MVP Definition v0.1

## هدف
Telegram Mini App + TON Blockchain
بازار: خلیج فارس (UAE, Kuwait, Qatar, Bahrain)

## داخل MVP
- ۱ نوع زمین: Common (Topsoil Flats)
- ۱ محصول: Lumen Apple (رشد ۴ ساعت)
- ۱ توکن: SAP (TON Jetton / TEP-74)
- زمین به‌عنوان NFT (TEP-62/64)
- Telegram Bot + Mini App
- custodial wallet پشت‌صحنه برای onboarding آسان

## خارج از MVP
- ROOT token
- زمین‌های Rare+
- Sproutlings / Golem-Hands
- Guild / DAO
- کارگاه‌ها

## استک
- Frontend: React 18 + TypeScript + TailwindCSS + @telegram-apps/sdk
- Contracts: Tact + TON Blueprint
- Bot: Python 3.13 + aiogram 3
- Network: TON testnet → mainnet
- Deploy: Vercel (frontend) + Railway/VPS (bot)

## معیار خروج
کاربر می‌تواند:
1. از طریق تلگرام وارد شود
2. یک زمین Common داشته باشد
3. Lumen Apple بکارد
4. پس از ۴ ساعت SAP برداشت کند
